xsd2thrift / xsd2proto by tranchis

This tool allows for converting XML Schema files (.xsd) to Thrift (.thrift) and Protocol Buffers (.proto).

Usage: java xsd2thrift.jar [--thrift] [--protobuf] [--output=FILENAME] [--package=NAME] filename.xsd --thrift : convert to Thrift --protobuf : convert to Protocol Buffers --output=FILENAME : store the result in FILENAME instead of standard output --package=NAME : set namespace/package of the output file

Install

No installation is required. The jarfile dist/xsd2thrift is autoexecutable. xsd2thrift has been tested on Java SE 6.
